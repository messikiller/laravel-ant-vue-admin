### Laravel-Ant-Vue-Admin

基于 Laravel + Vue Ant Deisgn 的后台管理系统脚手架，支持 Vue + Vue-router + Vuex

## 安装

```
git clone https://github.com/messikiller/laravel-ant-vue-admin.git
cd laravel-ant-vue-admin
composer install
npm install
npm run prop
```

Enjoy It !
